Regarde.tk

When in a call with my friends, I found it frustrating that there was no easy way to share my screen. Sure, I could ask my friend to download and install software, enter my connection ID, etc, but I knew there had to be a better way. I created Regarde to solve that issue. Regarde allows users to share their screen with others without the need to install any software. The best part, it's completely secure and peer to peer. By using WebRTC I am able to cut bandwith costs by 85%! Try it out!

Screenur

There are thousands of screenshot tools out there. What makes Screenur any different? Simply put, it's extremely quick. Pressing CTRL+SHIFT+3 immediately takes a screenshot, compresses the image, uploads it to imgur.com, copies the url to your clipboard, and opens the url in your browser. Everything is automatic. Pressing CTRL+SHIFT+4 does the same thing but allows you to select a region. It's a very specialized screenshot tool written in C++ using the Win32 API, GDI+, and libcurl libraries. Best of all, there's no install required. Try it out and see what you've been missing :)

Shortpoll

Shortpoll is a realtime polling app I worked on over the summer of 2016. It allows users to create a poll, which generates a poll url that can be visited by any device. As other users as the poll, results are updated in realtime (using websockets and redis) and these results can be seen as a bar graph and pie chart in realtime (more graphs planned in the future!). It is written in Nodejs, and code is available on Github.
